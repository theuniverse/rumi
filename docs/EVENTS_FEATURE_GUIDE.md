# Rumi Events 功能使用指南

## 功能概述

Rumi Events 集成了 Scraper 系统,可以根据你关注的艺术家(People)和场馆(Places)自动推荐相关活动。

## 核心功能

### 1. 实体关联 (Entity Linking)

#### People 页面 - 关联艺术家
1. 进入 People 页面
2. 点击任意艺术家进入编辑模式
3. 在编辑区域找到 "Link to Scraper" 部分
4. **智能建议**: 系统会根据艺术家名称自动推荐匹配的 Scraper 艺术家
   - 点击建议的蓝色按钮快速关联
   - 或使用下拉菜单手动选择
5. 保存更改

#### Places 页面 - 关联场馆
1. 进入 Places 页面
2. 点击任意场馆进入编辑模式
3. 在编辑区域找到 "Link to Scraper" 部分
4. **智能建议**: 系统会根据场馆名称自动推荐匹配的 Scraper 场馆
   - 点击建议的紫色按钮快速关联
   - 或使用下拉菜单手动选择
5. 保存更改

### 2. 关注功能 (Follow)

- **People**: 点击艺术家行右侧的星标图标 ⭐ 关注/取消关注
- **Places**: 点击场馆行右侧的星标图标 ⭐ 关注/取消关注
- 只有关注的实体才会用于活动推荐

### 3. Events 页面

#### 两个标签页

**My Events (我的活动)**
- 显示你标记为"要参加"的活动
- 可以切换活动状态:
  - Interested (感兴趣)
  - Attended (已参加)
  - Skipped (跳过)
- 显示活动的 lineup (阵容)

**Recommended (推荐活动)**
- 基于你关注的艺术家和场馆推荐活动
- 显示匹配原因标签:
  - 🎵 Artist Match (艺术家匹配)
  - 🎭 Venue Match (场馆匹配)
  - 📅 Date (日期相关)
- 点击 "I'm Attending" 按钮将活动添加到 My Events
- 点击 "Refresh" 按钮刷新推荐

## 推荐算法

活动推荐基于以下评分系统:

1. **艺术家匹配** (3分): 活动阵容中包含你关注的艺术家
2. **场馆匹配** (2分): 活动在你关注的场馆举办
3. **日期接近度** (1-2分): 活动日期越近分数越高
4. **置信度** (0-1分): Scraper 数据的置信度

活动按总分排序,分数越高越靠前。

## 使用流程示例

### 完整流程
1. **添加艺术家**: 在 People 页面添加你喜欢的 DJ/艺术家
2. **关联 Scraper**: 使用智能建议或手动选择关联到 Scraper 数据库
3. **关注艺术家**: 点击星标图标关注
4. **添加场馆**: 在 Places 页面添加常去的场馆
5. **关联 Scraper**: 关联到 Scraper 场馆数据
6. **关注场馆**: 点击星标图标关注
7. **查看推荐**: 进入 Events 页面的 Recommended 标签
8. **标记参加**: 点击感兴趣活动的 "I'm Attending" 按钮
9. **管理活动**: 在 My Events 标签管理你的活动

### 快速开始
1. 在 People 页面关注几个艺术家并关联 Scraper
2. 进入 Events 页面查看推荐
3. 点击 "I'm Attending" 添加到我的活动

## 数据来源

- **Scraper RefData**: 艺术家和场馆的参考数据
- **Scraper Events**: 从各种来源抓取的活动信息
- **Rumi Database**: 你的个人数据(关注、标记等)

## 注意事项

1. **首次使用**: 需要先关联至少一个艺术家或场馆才能看到推荐
2. **数据更新**: Scraper 数据定期更新,点击 Refresh 获取最新推荐
3. **匹配精度**: 智能建议基于名称相似度,可能需要手动确认
4. **活动时间**: 只显示未来的活动,过期活动会自动过滤

## 故障排除

### 没有推荐显示
- 确认已关注至少一个艺术家或场馆
- 确认关注的实体已关联到 Scraper
- 点击 Refresh 按钮刷新推荐
- 检查 Scraper 服务是否正常运行

### 智能建议不准确
- 使用下拉菜单手动搜索和选择
- 确保艺术家/场馆名称拼写正确

### 活动重复
- 系统会自动去重,如果仍有重复请报告 bug
